🗣️ Speaking Test (In-person, 15-20 minutes)

The speaking test is conducted face-to-face. Topics include:

  • Introducing yourself (rank, duties, experience)
  • Describing a safety drill procedure
  • Giving navigational commands
  • Reporting an incident (man overboard, engine trouble)
  • Using Standard Marine Communication Phrases (SMCP)
